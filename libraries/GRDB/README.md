# SwiftBindings.GRDB

.NET bindings for the [GRDB](https://github.com/groue/GRDB.swift) Swift SQLite toolkit, generated by the Swift/.NET binding generator.

## Usage

```csharp
using GRDB;

// Access GRDB database types
var config = new Configuration();
```

## Requirements

- .NET 10.0+
- iOS 13.0+
- Apple Silicon (arm64)
