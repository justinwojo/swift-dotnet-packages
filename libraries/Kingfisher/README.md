# SwiftBindings.Kingfisher

.NET bindings for the [Kingfisher](https://github.com/onevcat/Kingfisher) image downloading and caching library, generated by the Swift/.NET binding generator.

## Usage

```csharp
using Kingfisher;

// Access the shared manager
var manager = KingfisherManager.Shared;
var cache = ImageCache.Default;
```

## Requirements

- .NET 10.0+
- iOS 13.0+
- Apple Silicon (arm64)
