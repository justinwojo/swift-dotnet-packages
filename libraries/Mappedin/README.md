# SwiftBindings.Mappedin

.NET bindings for the [Mappedin](https://www.mappedin.com/) indoor mapping SDK, generated by the Swift/.NET binding generator.

## Usage

```csharp
using Mappedin;

// Access Mappedin indoor mapping types
var data = new MPIData();
```

## Requirements

- .NET 10.0+
- iOS 15.0+
- Apple Silicon (arm64)
