# SwiftBindings.Stripe.ThreeDS2

Transitive support package for the [Stripe iOS SDK](https://github.com/stripe/stripe-ios) bindings. Wraps `Stripe3DS2.xcframework` — an ObjC-only framework Stripe ships to power 3-D Secure 2 challenge flows used by `StripePayments`, `StripePaymentSheet`, `StripePaymentsUI`, `StripeIssuing`, and the `Stripe` umbrella.

> **You should not install this package directly.** It is pulled in automatically as a transitive dependency by the Stripe binding packages that need it. It exists as a standalone NuGet only so its binary can be shared across siblings without duplication.

## How It Works

These bindings are auto-generated by [Swift Dotnet Bindings](https://github.com/justinwojo/swift-dotnet-bindings) using .NET 10's native Swift interop via the [SwiftBindings SDK](https://www.nuget.org/packages/SwiftBindings.Sdk).
