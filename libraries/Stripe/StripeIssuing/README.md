# SwiftBindings.Stripe.Issuing

Native Swift interop bindings for `StripeIssuing` — the [Stripe Issuing](https://docs.stripe.com/issuing) push-provisioning module of the [iOS SDK](https://github.com/stripe/stripe-ios). Adds Stripe-issued cards to Apple Wallet via PassKit's in-app provisioning flow. These are not Objective-C proxy wrappers — they use .NET 10's native Swift interop for direct, high-performance calls into Swift APIs from C#.

## Installation

```
dotnet add package SwiftBindings.Stripe.Issuing
```

## Requirements

- .NET 10.0+
- iOS 15.0+
- macOS host for development
- A Stripe account with Issuing + push-provisioning enabled
- Apple Wallet entitlement (`com.apple.developer.payment-pass-provisioning`) granted by Apple

## Usage

```csharp
using StripeIssuing;

// Build push-provisioning details from your backend's response
var details = new STPPushProvisioningDetailsParams(
    cardId: "card_123",
    certificates: certs,        // byte[] / Data array from PassKit
    nonce: nonce,
    nonceSignature: nonceSignature);

// Pass `details` into your STPPushProvisioningContext flow.
```
