# Swift.StripeApplePay

.NET bindings for [StripeApplePay](https://github.com/stripe/stripe-ios.git), a Swift library.

## Installation

```
dotnet add package Swift.StripeApplePay
```

## Requirements

- .NET 10.0+
- iOS 15.0+
- Apple platform (macOS host for development)

## Building from Source

```bash
# 1. Build the xcframework from SPM
./build-xcframework.sh

# 2. Build the package (SDK generates bindings automatically)
dotnet build
```
